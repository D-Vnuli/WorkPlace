# Network Monitor

Network Monitor – настольное приложение для мониторинга сетевых устройств в локальной сети с удобным графическим интерфейсом.

Программа позволяет контролировать доступность оборудования по IP-адресам, получать уведомления о потере и восстановлении связи, вести журнал событий и организовывать устройства в древовидную структуру для удобного управления инфраструктурой.

Приложение разработано как практический инструмент для системных администраторов, IT-специалистов, офисов, ресторанов, гостиниц и других объектов, где требуется оперативное обнаружение недоступного оборудования.

---

## Возможности

### Мониторинг устройств

* Проверка доступности устройств по ICMP (Ping)
* Автоматический контроль состояния оборудования
* Обнаружение потери связи
* Обнаружение восстановления связи
* Настраиваемая задержка перед фиксацией отказа
* Отображение текущего статуса устройств:

  * 🟢 В сети
  * 🔴 Не в сети
  * ⚪ Нет данных

---

### Организация инфраструктуры

* Создание групп и подгрупп
* Неограниченная вложенность веток
* Древовидная структура устройств
* Перетаскивание устройств между группами
* Визуальное выделение проблемных веток
* Отображение родительских групп для устройств

Пример структуры:

```text
📁 Restaurant
 ├─ 📁 NVR
 │   ├─ Camera 1
 │   ├─ Camera 2
 │   └─ Camera 3
 └─ POS Terminal
```

---

### Уведомления

* Всплывающие уведомления при потере связи
* Уведомления при восстановлении связи
* Отображение полной ветки устройства
* Группировка нескольких аварий в одном окне
* Звуковое оповещение
* Работа через системный трей
* Модальные уведомления, требующие подтверждения пользователя

---

### Журнал событий

* Автоматическая регистрация событий
* Фиксация потери связи
* Фиксация восстановления связи
* Отображение времени события
* Отображение полной ветки устройства
* Просмотр истории изменений
* Очистка журнала через интерфейс

---

### Контроль данных

* Проверка на дублирование IP-адресов
* Защита от случайного добавления одинаковых устройств
* Проверка при изменении свойств устройства
* Исключение повторяющихся записей оборудования

---

### Интерфейс

* Светлая тема оформления
* Тёмная тема оформления
* Сохранение пользовательских настроек
* Сохранение размеров окна
* Сохранение структуры дерева устройств
* Современный интерфейс на PyQt6

---

### Работа с базой данных

* Хранение данных в SQLite
* Возможность выбора расположения базы
* Экспорт базы данных
* Поддержка общей базы данных для нескольких пользователей
* Автоматическое обновление данных при совместной работе

---

### Системный трей

* Сворачивание приложения в трей
* Работа в фоновом режиме
* Быстрое восстановление окна
* Уведомления через системный трей Windows

## Сборка EXE

Используется PyInstaller:

```bash
python -m PyInstaller --clean --onefile --windowed --name NetworkMonitor --icon=icons/icon-monitor.ico --add-data "icons;icons" main.py
```

После сборки исполняемый файл будет расположен в каталоге:

```text
dist/
```
