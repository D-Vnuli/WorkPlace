# Network Monitor

Network Monitor – приложение для мониторинга сетевых устройств в локальной сети с удобным графическим интерфейсом.

Программа позволяет отслеживать доступность устройств по IP-адресу, получать уведомления о потере связи, вести журнал событий и организовывать устройства в группы для удобного управления.

---

## Основные возможности

### Мониторинг устройств

* Проверка доступности устройств по ICMP (Ping)
* Автоматический контроль состояния устройств
* Отображение текущего статуса:

  * 🟢 В сети
  * 🔴 Не в сети
  * ⚪ Нет данных

### Группировка устройств

* Создание групп и подгрупп
* Древовидная структура устройств
* Перетаскивание устройств между группами
* Визуальное выделение проблемных веток

### Уведомления

* Всплывающие уведомления при потере связи
* Уведомления при восстановлении связи
* Отображение имени устройства и IP-адреса

### Журнал событий

* Регистрация всех изменений состояния
* Просмотр истории событий
* Очистка журнала через интерфейс

### Настройки

* Светлая и тёмная тема оформления
* Автозагрузка приложения при запуске Windows
* Выбор расположения базы данных
* Экспорт базы данных

### Системный трей

* Сворачивание в системный трей
* Быстрое восстановление окна
* Работа в фоновом режиме

### Обновление приложения

Поддерживается механизм безопасного обновления через файл:

```text
update.flag
```

При обнаружении файла приложение корректно завершает работу, что позволяет заменить исполняемый файл новой версией без вмешательства пользователя.

## Сборка EXE

Используется PyInstaller:

```bash
pyinstaller --onefile --windowed --name NetworkMonitor --icon=icons/icon-monitor.ico main.py
```

Готовый исполняемый файл будет создан в папке:

```text
dist/
```
